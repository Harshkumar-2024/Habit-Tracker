## Habit Tracker 📝✨

A full-stack Habit Tracker Web Application built with React, Node.js, Express, MongoDB, and OAuth authentication that helps users build consistency by tracking daily habits, monitoring streaks, and visualizing progress.

🚀 Features
🔐 Secure Google OAuth Login
✅ Create, update, and delete habits
📅 Daily habit completion tracking
🔥 Streak system to maintain consistency
📊 Progress visualization dashboard
☁️ MongoDB database for persistent storage
🎨 Modern responsive UI with smooth UX
⚡ Fast full-stack architecture using REST APIs
🛠️ Tech Stack

# Frontend

React
TypeScript
Tailwind CSS
Axios
React Router

# Backend

Node.js
Express.js
MongoDB
Mongoose
JWT / OAuth
REST API
# 📌 Purpose

The goal of this project is to help users develop positive habits and stay disciplined through streak-based motivation and progress tracking.
